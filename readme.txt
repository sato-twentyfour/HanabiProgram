起動方法
・コマンドラインからsample_server.pyを実行
・実行時引数はseed(任意の数)とagentmode(0は従来の決定論,1は行動の優先順位に応じて思考時間を変化させるエージェント)
・実行後はclient.htmlを起動するとゲーム開始
注意.事前にWebsocketのモジュールをインストールする必要があります





2019/10/25ごろ
このGUIバージョンのエージェントについてです
Playerクラスを継承していることやそれにより使用できるメンバ変数とメソッドに変更はありませんが，
クライアントとのやり取りをする関係上，choice_act()メソッドの返り値にエージェントのアクションを追加していただくようお願いします。
（送りましたファイルのDecideAgentに追加をしてあるので，そちらを見るほうが早いかもしれないです）

返り値は3文字のstringで
・エージェントがプレイするとき
p(プレイするカードの色)（プレイするカードの数字）

例：W1を出すとき
return pW1
となるように返り値を設定

・エージェントが手札を捨てる時
d(捨てるカードの色)（捨てるカードの数字）

例：W1を捨てる時
return dW1


・エージェントが情報を教える時
t(情報を教える相手(player_index))（教える色か数字）

例１：プレイヤー１に数字１を教えるとき
return t11

例２：プレイヤー１に色Wを教える時
return t1W

↑色は大文字でお願いします



エージェントを使うときのコードの変更点はCUIの時と同じですが，現時点では
プレイヤー(player_indexが1)先攻，エージェント(player_indexが2)後攻のみでしか使用できません。
必要であれば両方プレイヤー，先攻後攻逆ならすぐできると思います。
GUIでのオート対戦は少し時間がかかりそうです。

よろしくお願いします
